Weather API Web App ☁️
A full-stack weather application that fetches real-time weather data for any city using a Java backend and a clean, responsive frontend.

✨ Features
🔍 Search weather by city name
🌡️ Displays temperature, humidity, wind speed, and conditions
🌐 Connects to a live weather REST API
📱 Responsive layout — works on desktop and mobile
⚡ Java backend handles API calls and serves data to the frontend.

🚀 Getting Started

Prerequisites


Java 11+
A free API key from OpenWeatherMap (or your chosen weather API)


Setup

bash# Clone the repository
git clone https://github.com/Aman-kSingh/Api-projects.git
cd Api-projects

# Add your API key in the Java source file (src/main/...)
# Look for: API_KEY = "your-key-here"

Run

Open the project in Eclipse or IntelliJ → Run as Java Application → open index.html in your browser.
